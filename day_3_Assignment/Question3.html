data types in javascript

  1. String

    2. every variable values in "" is string

    var greeting="hello";

    hello is a string stored in greeting variable

  2. Number

    3. every integer value comes in number data type

    var age=20;

    20 is a number which stored in variable age

  3. Boolean

    4. true(1) and false(0) values are boolean values

    var alive=true; or var alive=1;

    in this there is only two options if it is true that is alive if false that means no same with 1 & 0

  4. Undefined

    5. undeclared variables default value is undefined

    var profit;

    here we declare a variable profit but not initalize with value so data type of this variable is undefined

  5. Null

    6. null value is not equal to 0 ie: it indicates the value of a variable is nothing(null)

    var profit=null;

    now value of profit is not 0 or not any other value, because value is null

  6. array

    7. collection of datatypes in [] is array

    var fruits=["apple", "mango", "banana"];

    fruits is array which contains more than 1 values

  7. object

    8. value and key pairs are called object which comes under {} like dictionary in python

    var person={name:"alice", age:20, gender:male}

    person is a object here and it has three key name age and gender with their respective values
